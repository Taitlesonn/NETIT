Enabling Accessibility Features

The GNOME desktop includes several features that enable people with dis-
abilities to operate applications on the desktop using alternative methods. 
These features include screen magnifiers (which enlarge areas of the screen), 
screen readers (which read text on the screen), and keyboard and mouse 
assistive features (such as sticky keys and slow mouse clicks). You can 
enable individual features as necessary.

Before you can use the individual accessibility features, you need to enable 
the accessibility features in GNOME. Follow these steps to enable the assis-
tive technologies packages:
