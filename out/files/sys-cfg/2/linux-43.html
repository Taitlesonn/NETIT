  If you have more than one hard drive in your computer, 

FDISK

  Here’s how to use the fifth menu choice to change to another disk:

 

a. Type 

5

 

and press Enter.

  

The 

FDISK

 screen displays all the hard drives in your system.

 

b. Type the number of the disk you want and then press Enter.

 

 The top of the 

FDISK

 menu screen displays the number of the 

drive that 

FDISK

 is working with.
