Opening an existing file in the Database Wizard

If you have an already existing Microsoft Access file that you want to open in 
OpenOffice.org Base, do the following from the initial Database Wizard:

 

1.  Select Open an Existing Database File.

 

2.  Select a file.

 

  You have these options:

 • 

Recently Used:

 If the file is in the Recently Used drop-down list, 

select the file there. Click Finish, and you’re done opening the file.

 • 

Open button

: If the file you want isn’t in the Recently Used drop-

down list, click the Open button and proceed to Step 3.
