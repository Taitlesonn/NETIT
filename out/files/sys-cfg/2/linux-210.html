3.  Modify the settings in the Modify Account dialog box and click OK.

 

  The Modify Account dialog box contains three tabs:

 • 

General:

 Contains general account information, such as the ISP 

server hostname, TCP port used, and login information.

 • 

Advanced:

 Contains settings for how long to leave messages on the 

ISP server.

 • 

Security:

 Contains settings for the encryption and authentication 

methods used by the ISP server.

 

4.  Select the Sending tab in the Accounts page to view SMTP server set-

tings; then select the e-mail account used for sending mail, and click 
the Modify button.

 

  If your Linux distribution configures a local e-mail server program, you 

may see another option for local mail delivery here. Make sure you 
select the entry that corresponds to your ISP account. You see the
