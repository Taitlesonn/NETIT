The GIMP is an incredibly complex program, with entire books written for 
the people who really want to use it heavily. Here are a few things to get you 
started.

The GIMP basic tools

The beauty of The GIMP is that all the tools appear in separate dialog boxes, 
allowing you to lay out multiple tools as you draw. The main dialog box 
allows you to coordinate between the different tool dialog boxes while edit-
ing an image.
