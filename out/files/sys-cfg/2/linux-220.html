Editing Text in the KDE Desktop: KWrite

Not to be left out, the KDE environment also provides a way to graphically 
edit text files. The basic editor for the KDE environment is KWrite. It provides 
simple word-processing-style text editing, along with support for code-syntax 
highlighting and editing.

To open KWrite, select KWrite from the KDE menu system on your desktop.
