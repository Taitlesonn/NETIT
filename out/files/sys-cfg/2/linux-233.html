Along the top of the window is the menu bar. OpenOffice.org Calc has all 
the features you expect from a modern spreadsheet system. It has too many 
menu options to cover in depth, so, instead, here’s a (non-exhaustive) sum-
mary of what you find on each menu:

 ✓ 

File:

 The usual Open, Save, Save As, Print, and Print Preview (under the 

term Page Preview) commands; a set of wizards; plus the capabilities to 
send documents through e-mail, create templates, and create Web pages

 ✓ 

Edit:

 The usual Select All and Find commands, along with change track-

ing, headers and footers, and plugin loading

 ✓ 

View:

 The usual Zoom functions and toolbars, along with the options for 

showing or hiding column and row headers
