  The Ubuntu disk partition window starts out with at least two possible 

selections:

 • 

A guided partition to install Ubuntu on the entire hard drive: 

By 

default, Ubuntu offers a guided partition where it automatically 
reformats the entire hard drive on the system for Ubuntu. If you 
want to run an Ubuntu-only workstation, this is the quickest and 
easiest way to go. If you select this method, skip to Step 11.

 

 Be careful — this is the default installation option! When you 

choose to install Ubuntu on the entire hard drive, any previously 
installed operating system (and data) will be removed. Be sure to 
back up any important files you have before you do the Ubuntu 
installation.

 • 

A manual partition to create your own partitions: 

If you select the 

manual partition process, Ubuntu turns control of the partition 
process over to you. It provides a partition utility, shown in Figure 
3-3, for you to create, edit, or delete hard-drive partitions. The 
manual partition utility displays the current hard drives, along 
with any existing partitions configured in them. Each hard drive 
appears as a separate item in the partition listing and is assigned a 
device name (such as 

sda

 for the first hard drive and 

sdb

 for the 

second hard drive). You can manually remove, modify, or create 
individual partitions on any hard drives installed on the system.
