System Monitor Process Table Columns

Column

Description

Process Name

The program name of the running process.

User

The owner of the process.

Status

The status (either sleeping or running) of the process.

Virtual Memory

The amount of virtual system memory allocated for the 
process.

Resident Memory

The amount of physical memory allocated for the process.

Writable Memory

The amount of memory allocated to the process currently 
loaded into physical memory (active).

Shared Memory

The amount of memory shared between this process and 
other processes.

X-Server Memory

The amount of memory the process shares with the X
