 Chapter 21: Ten Steps to Setting Up a Samba Server

 

3.  Click the Add User button to add a new Samba user.

 

  The Create New Samba User window appears. This window contains 

four areas where you have to enter specific information:

 • 

UNIX Username: 

A drop-down list where you can select the Linux 

user account to map.

 • 

Windows Username:

 A text box for typing the associated Windows 

network username that the remote client will use.

 • 

Samba Password:

 Here’s where you enter the password assigned 

to the Windows network username (remember, this can be — and 
often is — different from the Linux password).

 • 

Confirm Samba Password:

 Here’s where you retype the Samba pass-

word to ensure that you typed it correctly.

 

  You must match the Windows network username with the Linux user-

name. In the example shown in Figure 21-8, Samba maps the Windows 
network username 

fflinsto

 to the Linux user account 

fred

.
