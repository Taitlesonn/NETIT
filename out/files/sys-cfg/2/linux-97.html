Interface

The Interface tab provides a few settings for customizing the way menus and 
toolbars appear in windows:

 ✓ 

Show icons in menus:

 Displays icons in the menu alongside the text 

menu item.

 ✓ 

Editable menu shortcut keys:

 Enables using shortcut keystrokes to 

select menu options.

 ✓ 

Toolbar button labels:

 Sets how toolbar items appear: text only, icon 

only, icon with next below, or icon with text next to it.

These window settings apply to any window that uses the GNOME interface. 
They don’t apply to applications that don’t use the GNOME library, such as 
applications built on the KDE desktop library.
