System

The System menu provides access to your system configurations. As a gen-
eral rule, GNOME provides two types of configuration settings:

 ✓ 

User-specific settings:

 Items that only affect the way your user account 

operates on the system, such as screen resolution, background colors, 
and keyboard preferences.

 ✓ 

System-specific settings:

 Items that affect the entire Linux system, such 

as network settings, sound settings, and user accounts.

Table 4-3 details how these settings appear on the System menu.
