Playing 3-D Games

One of the strengths of Windows PCs has been the support for advanced 
video games. These games utilize high-end graphics that often require spe-
cialized 3-D video cards to operate.

Again, this is another area where Linux had never quite caught up to 
Windows, until now. Two major 3-D video card vendors, ATI and NVIDIA, 
have released Linux drivers for their products, allowing game developers to 
enter the Linux market.

There’s a catch to this, though: Both ATI and NVIDIA released Linux binary 
drivers for their 3-D video cards, but not the source code. A true open-
source system must include source code for all the components included in 
the system. This has caused quite a dilemma for Linux distributions.

For a Linux distribution to include the ATI or NVIDIA binary drivers violates 
the true spirit of open-source software. But if a Linux distribution doesn’t 
provide these drivers, it risks falling behind in the Linux distribution wars 
and losing market share.

notifying you that it installed a driver for a proprietary card. You can view
