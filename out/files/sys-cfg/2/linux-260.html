 Chapter 12: Word Processing and More with OpenOffice.org

OpenOffice.org Math supports (among others) its own OpenDocument 
format (ODF), the equivalent StarOffice format (SXM) and MathML 1.01 
(MML), which isn’t a program. 

MathML

 is a standard — similar to HTML for 

working on the Web — specifically it’s version 1.01 of this standard. For this 
particular version of MathML, see 

www.w3.org/TR/REC-MathML

. The main 

standard page is available at 

www.w3.org/Math/

. If you need to add formu-

las to Web pages, this site can be quite an interesting read!
