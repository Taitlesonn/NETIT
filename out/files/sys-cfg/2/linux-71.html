  The default setup in Fedora is to create a Logical Volume Manager 

(LVM). The LVM controls the hard-drive partitions as a logical unit, 
allowing you to easily add partitions into the same LVM unit. With this 
method you can easily expand disk space on your system without 
having to reconfigure anything.

 

11.  Select the location for the boot loader, and click Next.

 

  Fedora allows you to customize the Grand Unified Bootloader (GRUB) 

by selecting where to place it as well as by selecting which operating 
systems to include in the boot menu. This is what allows you to boot 
Fedora or Windows from your hard drive.
