9.  Review the settings for the new virtual disk drive file, then click the 

Finish button to create the file.

 

  The wizard creates the file, and then it returns you to the original Virtual 

Server Wizard process.

 

10.  Ensure that the new virtual disk file appears as the selected virtual 

disk, and then click the Next button.

 

11.  Review the settings you selected for the virtual server, and then click 

the Finish button to create the virtual server.
