Working with the Sandbox

When you have your virtual server running, you need to become familiar with 
a few VirtualBox commands.

When you first start your virtual server, a set of icons appears at the bottom 
of the window (refer to Figure 20-8). These icons indicate when the virtual 
server is accessing hardware on the system. The icons represent

 ✓ 

The hard drive

 ✓ 

The CD/DVD-ROM drive

 ✓ 

The floppy drive

 ✓ 

The network interface

 ✓ 

The shared folder

The bottom-right corner of the window also contains a mouse icon, a key 
icon, and the phrase 

Right Ctrl.

 When you start the virtual server, VirtualBox 

takes control of the keyboard and mouse on the host system, and then it 
turns them over to the virtual server.
