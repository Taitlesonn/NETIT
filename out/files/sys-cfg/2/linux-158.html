Setting up KPPP

The KPPP package requires about the same amount of work as the GNOME-
PPP package for connecting to the Internet. After installing the KPPP package, 
choose Applications

➪

Internet (Fedora labels it as Internet Dial-Up Tool) to 

open it. The main KPPP dialog box is shown in Figure 8-3.

 

Figure 8-3:

 

The KPPP 

main dialog 

box.

 

Here’s how to set up KPPP for your ISP account:

 

1.  Click the Configure button to change the default settings.

 

  The KPPP Configuration dialog box has four tabs for configuring your 

dialup connection:

 • 

Accounts:

 Defines the logon information for your ISP

 • 

Modems:

 Defines your modem properties

 • 

Graph:

 Defines how to display connection statistics

 • 

Misc:

 Defines various behaviors for the KPPP application

 

2.  Click the Modems tab to set up your modem.
