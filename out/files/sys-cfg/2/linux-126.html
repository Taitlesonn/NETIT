Text Color

.

) in Linux.

A

SUID

) executable file, which uses a different user

A SetGID (SGID) executable file, which uses a different

White text with a 
red background

A 

broken link

 file.

Yellow text with a

A 

device

 file (basically, the file that is used to talk to a piece

Black text with a

A directory where you can only delete or change the files 
that you own or have 

write permissions

 for (see the section 

“Understanding file listing information” later in this chapter 
for more).

A lot of files begin with the dot in your home directory. Applications that allow 
you to customize features store those settings in hidden files in your home 
directory.
