Remove the selected text from the document and save it in

Make a copy of the selected document text and save it in

Paste

Place the text from memory in the document at the cursor’s

Undo

Undo the last change you made to the document. Click the 
down arrow to choose how far you want to back up to.

Restore

Reinstate the last change to the document after using Undo 
to cancel it. Click the down arrow to choose how far you 
want to redo.
