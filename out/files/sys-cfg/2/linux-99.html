GNOME enables the assistive technologies features when you log back in to 
the system. After you’ve enabled the assistive technologies features, you can 
configure the functions you require. There are three main categories:

 ✓ 

Preferred Applications:

 Allows you to define assistive applications, 

such as screen readers, that should start automatically at startup

 ✓ 

Keyboard Accessibility:

 Defines keyboard features, such as 

sticky keys

 

to simulate multiple key presses and 

slow keys

 to ignore random key 

presses.

 ✓ 

Mouse Accessibility:

 Defines mouse features, such as simulating 

double clicks and mouse 

dwells

 (hovering the mouse over an item 

without clicking it).
